Книга контактов API

Express-приложение с маршрутами, middleware и статикой.

Быстрый старт

  1. Запустите сервер: npm install и npm start.
  2. API база: /api/contacts, статус: /api/health.
  3. Статическая папка доступна прямо с этого домена.

Примеры запросов

GET  /api/contacts
GET  /api/contacts?search=alex
GET  /api/contacts/c2
POST /api/contacts
body: { "name": "New User", "phone": "+7 900 000-00-00", "email": "new@ex.com" }
